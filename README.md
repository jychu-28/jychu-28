## 📌 Current Focus

I am currently working on genome collinearity visualization strategies for ultra-large plant genomes, especially using peony genomes as an example.

The goal is to make chromosome-scale comparative genomics more practical by splitting oversized chromosomes into chunks, restoring coordinates, detecting structural variation with SyRI, and visualizing results with plotsr.
